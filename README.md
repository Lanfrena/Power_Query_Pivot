# Power_Query_Pivot

### Цель

1. Загрузить и обработать таблицы с показателями факта и планов продаж через Power Query

2. Добавить таблицы в модель данных  

3. Создать в модели данных связи для таблиц план-факт и меры расчета:
   - меру факта продаж Товарной группы1 за август 2022 (кол-во)
   - меру % выполнения плана продаж (кол-во)
   - меру расчета выручки (USD)
   - меру расчета валовой маржинальной прибыли (USD)
   - структуру продаж по Брендам (%)
   - средний курс продаж (USD)

4. Вставить сводную таблицу из модели данных в разрезе Брендов:
- с % выполнения ПЛАН-ФАКТ (кол-во)
- Суммой Валовой маржинальной прибыли (USD)
  
5. Вставить сводную диаграмму из модели данных в разрезе Брендов и дат:
- с % выполнения ПЛАН-ФАКТ (кол-во)
